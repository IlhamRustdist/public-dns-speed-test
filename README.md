# Public DNS Speed Test

Browser-based DNS over HTTPS (DoH) speed testing tool.

## Features

- Real-time DNS latency measurement
- Multiple public DNS providers
- Automatic ranking
- No backend required
- Works on GitHub Pages

## How It Works

This tool measures DNS resolution speed using DNS-over-HTTPS (DoH)
directly from your browser.

Results depend on your ISP and location.

## Supported Providers

- Cloudflare
- Google
- AdGuard
- Quad9
